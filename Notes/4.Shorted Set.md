
## Shorted Set in Redis

### ZADD

```sh
ZADD myset 1 "one"
ZADD myset 2 "two"
ZADD myset 3 "three"
```

### ZRANG
```sh
ZRANG myset 0 2
ZRANG myset 0 -1
```

### ZRANK TO find the rank(index) of the element 

```sh
ZRANK myset "one"
```

### ZREVREANK To Find Reverse Rank
```sh
ZREVRANK myset "one"
```
